## Алгоритм LZW сжатия данных
# Теоретическая часть
LZW (Lempel-Ziv-Welch) - алгоритм словарного сжатия, созданный в 1979 году. Используется в GIF, TIFF, PDF.

**Принцип работы:**
Инициализация словаря всеми возможными символами

Поиск самой длинной строки, уже находящейся в словаре

Добавление новой строки в словарь (текущая + следующий символ)

Вывод кода для найденной строки

**Преимущества:**
Не требует передачи словаря

Быстрое сжатие/распаковка

Хорошо сжимает тексты с повторениями

**Недостатки:**
Плохое сжатие случайных данных

Рост словаря требует управления
